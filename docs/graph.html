Graph¶

All nodes are one of the following types:

Node Types¶

APP
Application of a function or operator to a set of values.
OP
Operator a mapping between multiple operands returning a single result.
VAL
A value providing bytes.

Node Instances¶

The core graph node types in Blaze are:

  • App
  • ArrayNode
  • ExpressionNode
  • FloatNode
  • Fun
  • FunApp
  • IndexNode
  • IntNode
  • Literal
  • Op
  • Slice
  • StringNode

The domain and codomain of an operator are determined only in the context of an application binding.

The domain and codomain of an function are fixed by definition.

Holds the base classes for graph nodes.

class blaze.expr.graph.App(operator)¶

The application of an operand producing concrete values.

For example:

In[0]: a = 2 + 3

The resulting value of a is App( Op(+), 2, 3).

blaze.expr.graph.typeof(obj)¶

BlazeT value deconstructor, maps values to types. Only defined for Blaze types.

>>> typeof(IntNode(3))
int64
>>> typeof(Any())
top
>>> typeof(NDArray([1,2,3]))
dshape("3, int64")
