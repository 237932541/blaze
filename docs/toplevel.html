Toplevel¶

blaze.toplevel.fromiter(iterable, dshape, params=None)¶

Create an Array and fill it with values from iterable.

iterable : iterable object
An iterable object providing data for the carray.
dshape : str, blaze.dshape instance
Specifies the datashape of the outcome object. Only 1d shapes are supported right now. When the iterator should return an unknown number of items, a TypeVar can be used.
params : blaze.params object
Any parameter supported by the backend library.

out : an Array object.
